A) Preparing - top

Make sure you have the following:

  1. PHP 5.6.4 +
  2. MySQLi Extension
  3. GD Library
  4. And make sure you have VideoTube already installed on your site.

B) Installation Process - top

  1. Upload the folder "Red" on the FTP server and put inside the folder "templates" on your site. (usually located on public_html folder when installing it on a domain).

C) Configuration - top

  1. Log in with your admin Account. the go to yourdomain.com/admin
  2. in your admin panel go to the menu item "themes".
  3. click Select on the theme named "Red".
  4. click Save Settings and clear your browser cache.
